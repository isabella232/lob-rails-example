# Lob Ruby on Rails Postcard Examples

## Install

To insall the Ruby on Rails demo:

```
$ git clone git@github.com:lob/lob-rails-example.git
$ $ cd lob-rails-example/
$ bundle install
```

##Run

Start the rails server:
```
$ rails server
```

Now, visit the demo site in your browser:

```
http://localhost:3000/
```
